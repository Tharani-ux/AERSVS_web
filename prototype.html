User Manual: Assembling & Wiring Your PCB Circuit

1. Understanding the Components & Connections

  • ESP32-WROOM-32D - Microcontroller
  • MCP2515 CAN Module - CAN Communication
  • Iridium 9603N - Satellite Communication
  • Buttons (Medical, Fire, Police) - Input Controls
  • LEDs (Rx, Tx, Power) - Status Indicators
  • Power Supply (3.7V Li-ion & 5V Regulator)

2. Schematic Design in KiCad

Steps to create a schematic:

  1. Open KiCad and create a new project.
  2. Add components: ESP32, MCP2515, Iridium 9603N, etc.
  3. Connect components using correct wiring.
  4. Run an Electrical Rule Check (ERC).

3. PCB Layout in KiCad

Ensure correct footprints, routing, and spacing.

4. Firmware Development for ESP32

Use the following code snippet to initialize CAN communication:

#include <SPI.h> #include <Wire.h> #include <CAN.h> void setup() { Serial.begin(115200); if (!CAN.begin(500E3)) { Serial.println("CAN Init Failed!"); while (1); } Serial.println("CAN Initialized Successfully"); }

5. Final Testing & Debugging

Check LED responses, verify CAN communication, and test the Iridium modem.
