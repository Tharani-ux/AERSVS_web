Automatic Emergency Response System via Satellite (AERSVS)

Step 1: Automated & Manual Triggering

Scenario 1.1 - Automated Triggering (AT)

When a vehicle crashes, MEMS sensors detect the impact and trigger AERSVS within 10 to 30 milliseconds.

Process Explanation

  • MEMS sensors detect crashes via accelerometer, brake, and seatbelt sensors.
  • Data is transferred through CAN bus (digital and analog communication ports).

Setup 1: Microcontroller Integration

  • Microcontroller bridges the car's CAN network and Iridium modem.
  • Reads data from CAN bus and sends it via UART to Iridium.

Setup 2: CAN Bus Integration

  • Microcontroller connects to the CAN bus via a CAN transceiver.
  • CAN_H and CAN_L connect to the car's CAN bus.
  • Communication via TXD (CAN data to bus) and RXD (CAN data from bus) pins.

Setup 3: Iridium 9603 Modem Connection

  • Power: 3.2V-5.5V
  • UART Interface: TXD -> RX (MCU), RXD -> TX (MCU)
  • AT Commands: AT+SBDWT=<data>, AT+SBDIX

Data Flow Example

(Car CAN Bus) ----> [CAN Transceiver] ----> (Microcontroller)
                                     |
                              (UART Interface)
                                     |
                            [Iridium 9603 Modem]
